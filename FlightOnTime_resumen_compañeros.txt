FLIGHTONTIME – MVP (RESUMEN PARA COMPAÑEROS)

1. QUÉ ES EL PROYECTO
Proyecto de Machine Learning para predecir si un vuelo será:
- 0 = Puntual
- 1 = Retrasado

Es un MVP pensado para integrarse después en una API.

--------------------------------------------------

2. PROBLEMA QUE RESUELVE
Los retrasos en vuelos generan:
- molestias a pasajeros
- costes operativos
- problemas de planificación

El modelo permite anticipar retrasos y tomar decisiones antes del despegue.

--------------------------------------------------

3. DATOS UTILIZADOS
Dataset histórico de vuelos con clima integrado.

Variables usadas:
- hour → hora de salida
- distance → distancia del vuelo (km)
- marketing_airline_network → aerolínea
- temp_mean → temperatura media
- precipitation → precipitación
- wind_speed → velocidad del viento

Variable objetivo:
- target (0 puntual, 1 retrasado)

--------------------------------------------------

4. PROCESO DE TRABAJO (DATA SCIENCE)

1) Limpieza y exploración de datos
- revisión de nulos
- selección de variables relevantes

2) Feature engineering
- separación de variables numéricas y categóricas
- escalado de variables numéricas
- one-hot encoding para aerolínea

3) Modelado
Se entrenaron dos modelos:
- Logistic Regression (modelo final)
- Random Forest (modelo comparativo)

--------------------------------------------------

5. RESULTADOS DE LOS MODELOS

LOGISTIC REGRESSION (MODELO FINAL)
- Accuracy aproximada: 0.68 – 0.81 (según umbral)
- Modelo simple y explicable
- Fácil de integrar en backend
- Elegido para el MVP

RANDOM FOREST
- Accuracy aproximada: 0.81
- Mejor ROC AUC
- Menor interpretabilidad
- Usado solo como comparación

--------------------------------------------------

6. MEJORAS REALIZADAS

Mejora 1: Logistic Regression
- Ajuste del umbral de decisión
- Objetivo: aumentar la precision
- Resultado: menos falsas alarmas, menos recall

Mejora 2: Random Forest
- Ajuste de hiperparámetros
- Objetivo: reducir overfitting
- Resultado: modelo más estable

Las mejoras se usan solo para análisis, no se exportan.

--------------------------------------------------

7. MODELO EXPORTADO
Solo se exporta el modelo final base:
- Archivo: MVP_entrenamiento.pkl
- Incluye preprocesamiento + modelo entrenado
